Pipeline — Projects

Gesture App

Flagship

A real-time hand-gesture interface built with OpenCV and MediaPipe that replaces the mouse with hand tracking. DRAW mode turns finger position into a canvas — adjustable stroke thickness, a resizable eraser (thumb + index + middle), and a 7-swatch color strip selected with a 3-second index-finger hold. COUNT mode maps finger-count gestures to presentation controls — next slide, previous slide, start, stop — each confirmed with a hold gesture and audio feedback. Tuned MediaPipe's model complexity and confidence thresholds to keep it running smoothly in real time.
